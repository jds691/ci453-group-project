    Information Security

    Page contents

    What is Information Security?

    Information security is the practice of protecting information by getting rid of any risks. It protects personal information, financial information and sensitive information from unauthorised access. It is built around three main objectives which are known as the CIA, these stand for:

    1. Confidentiality - This means the information is not disclosed to unauthorised individuals.
    2. Integrity - This is so that the information maintains accurate and unchanged.
    3. Availability - This allows anyone to access it at anytime it is needed.

    What dangers are there?

    There are many security issues that come with information security. These can include:

    • Human error - People can make mistakes meaning information is at risk.
    • Cyber threats - These can include phishing, hackers and malware.
    • Having more and more mobile and internet of things (IoT) devices - This adds new security challenges as these devices can be lost or stolen.
    • Insider threats - Employees pose a risk as some may hold sensitive information.

    How to stop these risks
    • Training for employees - this means employees can identify threats and respond to them
    • Encryption - this protects sensitive information when it is sent or received
    • Firewalls - firewalls control access to computer networks and prevent unauthorised access to information
    • Penetration testing - this can be used to identify vulnerabilities in an organisation's systems

    Although there are many ways to improve information security, there are also a lot of challenges that come with it. There are always new cyber threats meaning the security you may already have will soon not be good enough and needs to constantly be improved. Integrating security into businesses is complicated as it requires a lot of time to do it correctly and this can often be rushed, meaning the business is more likely to have their security breached.
